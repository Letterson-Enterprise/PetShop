# PetShop
Sistema de petshop como projeto final da disciplina de backend

## Como rodar:
1. Backend: dotnet run em backend/ (confirme a porta — environment.ts usa https://localhost:5001/api; rode dotnet dev-certs https --trust)
2. Front: npm start em frontend/ → http://localhost:4200

## Para acessar:

Com a tela de login aberta, digite nos respectivos campos que encontram-se a seguir:

- Usuário: "admin"
- Senha: "administracao"
